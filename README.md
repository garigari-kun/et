## easy-tmux
Make tmux easier

## How to install
```
$ go get -u github.com/garigari-kun/et
```

## Usage
```
$ et
=====Create new session or Attach another session=====
0: Create New Session
1: bugbounty
2: easy-tmux (Attached)
3: shutto
4: some-server
======================================================
Enter what you want: 4

```

Typing ``` et ``` shows you choices you can choose.<br>
Simple.
