## easy-tmux
Make tmux easier

## How to install
```
$ go get -u github.com/garigari-kun/easy-tmux
```

## Usage
```
$ et
Create new session or Attach another session.
0: Create New Session
1: Work 
2: easy-tmux (Attached)
3: New feature fo golang 
Enter what you want:

```

Typing ``` et ``` shows you choices you can choose.<br>
Simple.
